# Inventory Management System

A simple **Inventory Management System** built in **Python** with **MongoDB** for storing product data.  
This project demonstrates the use of **Singleton design pattern**, basic CRUD operations, and database interaction with MongoDB.

---

## Features

- Add new products with name, price, and quantity
- Find a product by name
- Update product quantity
- Remove a product from inventory
- Singleton database connection (only one connection to MongoDB)
